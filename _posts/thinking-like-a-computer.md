In programming, one crucial thing to remember, before everything else, is to think like a computer.
In every line of code in your program, something happens. Just like a computer, do everything in steps.

If one line says:
```python
text = "Hello World"
```
then think that.

Then think of the next line:
```python
print(text)
```

and so on in small, discrete steps.
Follow every line like a stepping stone.

Doing so can save you a lot of testing your app, but make sure to test it every once in a while, frameworks
and operating systems can be quirkier than you expect.
